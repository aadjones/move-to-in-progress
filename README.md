# Move to In Progress

An interactive satire about modern project management tools.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to experience the app.

## How to Play

1. Drag the task card to "In Progress"
2. Follow the card as it falls
3. Try to complete the task
4. Good luck.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Tone.js (for audio - to be added)

## Development

```bash
npm run dev    # Start dev server
npm run build  # Build for production
npm run preview # Preview production build
```
